Bonjour

Ce TXT a pour but de vous indiquer les différentes commandes implémentées et leurs effets.
A des fins de tests, 5 formes ont été implémentées, réparties parmi deux groupes : 
groupe1 : rectangle1, rectangle2, cercle1, 
groupe2 : carre1, 
sans groupe : triangle1

Les commandes suivantes sont implémentées pour tester toutes les actions possibles : 

afficher_g1 : affiche le groupe1
afficher_g2 : affiche le groupe2
afficher_all : afficher tous les groupes

deplacer_r1 : déplace rectangle1 de 5000;5000
deplacer_r2 : déplace rectangle1 de -5000;-5000
deplacer g1 : déplace tout le groupe1, de 25000;25000

grouper_g2_t1 : ajoute triangle1 dans le groupe2
degrouper_cr1 : retire cercle1 du groupe2

positionner_r2 : place rectangle2 en -50000;-50000 quelles que soient ses coordonnées d'origine
